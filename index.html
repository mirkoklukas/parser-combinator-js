A Lisp Parser in JavaScript

Grammar:

(begin expr ...)
(define id expr)
(if cond succ alt)
(lambda (id ...) expr)
(set! id expr)

(+ expr expr ...)
(- expr expr ...)
(* expr expr ...)
(/ expr expr ...)
(= expr expr)

Example Code:

(begin
  (define x 2) 
  (define pow 
    (lambda (a b) 
      (if (= b 0) 
        1 
        (* a (pow a (- b 1)))))) 
  (pow x 4))
